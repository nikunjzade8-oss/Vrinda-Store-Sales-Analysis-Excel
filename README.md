# Vrinda-Store-Sales-Analysis-Excel
A comprehensive data analysis project using MS Excel to analyze sales performance of Vrinda Store in 2022, providing insights into customer demographics, top-performing states, and sales channels to drive business growth.
# Vrinda Store Sales Data Analysis (2022)

## Project Objective
The objective of this project is to analyze the sales data of Vrinda Store for the year 2022 to uncover key trends, customer behavior, and regional performance. This analysis aims to provide actionable insights that can help the store improve its marketing strategies and increase sales for the upcoming year.

## Dataset Overview
The dataset contains **31,000+ rows** of sales transactions, including:
- **Order Details:** Order ID, Date, Status (Delivered, Cancelled, etc.)
- **Customer Profile:** Gender, Age, Age Group
- **Product Details:** Category (Kurta, Set, Western Dress), Size, Quantity
- **Geography:** City, State (India)
- **Sales Channels:** Amazon, Myntra, Flipkart, Ajio, etc.

## Key Questions Answered
1. Compare Sales and Orders using a single chart.
2. Which month had the highest sales and orders?
3. Who purchased more: Men or Women?
4. What are the different order statuses in 2022?
5. List the top 5 states contributing to sales.
6. Relation between Age and Gender based on orders.
7. Which channel is contributing maximum to sales?

## Key Insights & Findings
- **Gender Performance:** Women are the primary customers, contributing approximately **64%** of total sales.
- **Top States:** Maharashtra, Karnataka, and Uttar Pradesh are the top 3 states, contributing significantly to the revenue.
- **Target Audience:** The **Adult age group (30-49 years)** is the highest contributing segment (~50%).
- **Top Channels:** **Amazon, Myntra, and Flipkart** account for nearly 80% of total sales.
- **Order Status:** About **92%** of orders were successfully delivered.

## Tools Used
- **Microsoft Excel:** Data Cleaning, Pivot Tables, Formulas (VLOOKUP, IF), and Dashboarding.

## How to Use
1. Download the `Vrinda Store Data Analysis.xlsx` file.
2. Open it in Microsoft Excel to view the raw data, processed sheets, and the final interactive dashboard.
